#### **Justification of Hours**

**Dominic Iquina:**
Dominic’s ~80 hours were heavily focused on backend architecture, database management, and system testing. During the first semester, his time was allocated to co-creating the initial project requirements, setting up the repository, and configuring the backend database schema. In the second semester, Dominic focused on making the database work with real data, handling the backend logic for the various attendance methods (Start, End, and Manual tracking), and ensuring secure data flow across the platform. His hours also include shared responsibilities in executing the system test plan and validating data accuracy.

**Mutazz Zeidan:**
Mutazz’s ~80 hours were primarily dedicated to the front-end implementation, user interface design, and system testing. During the first semester, Mutazz invested time in co-defining project requirements, wireframing the Flutter mobile application, and designing the core visual components. In the second semester, his activities shifted toward building out the interactive screens, refining the visual aesthetics to ensure the UI looked professional, and creating a smooth, functional user experience across the app. His hours also account for cross-platform UI widget testing and resolving front-end bugs.
